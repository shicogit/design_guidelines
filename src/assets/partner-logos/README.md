# Partner logos

Drop each partner's app/brand logo here as an **SVG** (preferred) or PNG, named by the partner
slug (lowercase, spaces → hyphens). The download reminder modal renders it at 26px; if a file is
missing, it falls back to a brand-color swatch.

Expected filenames:

- `xero.svg`
- `cfc.svg`
- `capital-one.svg`
- `paypal.svg`
- `gusto.svg`
- `us-bank.svg`
- `wafd.svg`
- `wells-fargo.svg`
- `adp.svg`

(Square / app-icon style works best at 26px.)
